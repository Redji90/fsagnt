# Telegram-бот для анализа фигурного катания

Telegram-бот с интеграцией LLM для анализа результатов соревнований по фигурному катанию и выдачи детальной аналитики.

## Возможности

- Анализ результатов соревнований по фигурному катанию
- Генерация аналитики с помощью LLM
- Детальная статистика по спортсменам
- Сравнение выступлений
- Прогнозирование результатов

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Redji90/fsagnt.git
cd fsagnt
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Заполните `.env` файл вашими токенами:
- `TELEGRAM_BOT_TOKEN` - токен вашего Telegram бота (получите у @BotFather)
- `OPENAI_API_KEY` - API ключ OpenAI (или другой LLM провайдер)

## Запуск

```bash
python bot.py
```

## Использование

После запуска бота в Telegram:
- `/start` - начать работу с ботом
- `/help` - список доступных команд
- `/analyze` - анализ результатов соревнований
- `/stats` - статистика по спортсменам
